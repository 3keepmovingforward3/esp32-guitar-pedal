# esp32-guitar-pedal
This project is a way to make an all purpose guitar pedal using an ESP32.

# Goals 
## Pedal Types
1. Distortion
2. Delay
3. Phaser
4. Tuner
5. Octaver
6. Pitch Shift
7. Harmonizer
8. Flanger
9. Chorus
10. Compressor
11. Noise Gate
12. Reverb
13. Looper

## Chaining
I'd like to be able to chain; this will be limited by the esp32 SOC.
